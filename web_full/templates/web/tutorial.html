    Instalación en Parrot-Sec ó Kali-Linux

    En primer lugar arrancamos nuestro sistema Linux iniciando sesión con el usuario que hemos creado al instalar nuestra distribución. Iniciamos el servicio “ssh” para poder conectarnos y comprobamos que esté activo.

    En LINUX no es necesaria la conexion a internet

    • 1.$ sudo service ssh start
    • 2.$ service ssh status

    Solo para trabajar remotamente

    Cuando arrancamos la aplicacion debemos introducir en server la dirección IP de nuestra máquina, que la averiguamos con el siguiente comando:

    • 1.$ ifconfig

    En el caso de tener Wifi sería wlan0, esto podría variar en el caso de usar una antena wifi externa.

    Si tuviéramos conexión por cable pondríamos la de eth0 En mi caso introduzco la wlan0 ya que estoy conectado por wifi.

    Cuando ya tenemos los servicios activos podemos descargar el programa.

    • 1.Linux
    • 2.Windows

    Con la ayuda del comando 'ls' visualizamos el nombre de la carpeta y nos movemos a ella con 'cd' donde está alojado el archivo compilado del proyecto.

    • 1.ls
    • 2.cd HashCrack/

    Aqui introducimos nuestro usuario y contraseña con los que iniciamos sesión y en server introducimos localhost ya que estamos trabajando en local.

    Si todo a ido bien nos saldrá este mensaje y tendremos acceso al menú del programa.

    Bienvenido al menú

    Aquí podemos elegir entre usar HashCrack para el crackeo de los hashes y TheHarvester para indexar correos, hosts y subdominios de un dominio en particular utilizando los principales motores de búsqueda (google, pgp, netcraft, twitter, etc).

    Aquí ya podriamos utilizar todas las funciones de nuestro programa que se explicarán en el manual de uso del programa. Paso a detallar como usarlo en windows. Para agregar el diccionario nos dirimos a la siguiente ruta:

    • 2.cd /usr/share/wordlists/
    • 2.nano list.txt

    Con el comando nano creamos el archivo y escribimos dentro las palabras a comprobar ó también podemos descargar un diccionario y renombrarlo como list.txt.

    Descargamos el winrar dirigiéndonos al gitlab del proyecto donde hay un enlace para descargarlo para windows.

    Extraemos la carpeta y ejecutamos el archivo.

    Aquí introducimos la ip que hemos obtenido con el paso 2.

    Una vez el inicio de sesión es correcto, podemos utilizar el software en windows de forma remota.
